AI-Based Smart Jewelry Theft Detection Using Computer Vision

📌 Project Overview
This project implements an AI-based real-time jewelry theft detection system using Computer Vision and YOLO object detection. The system monitors CCTV footage inside jewelry stores, detects jewelry items and human presence, tracks object movement, and generates alerts when suspicious activity is identified.

🎯 Problem Statement
Manual CCTV monitoring in jewelry stores is inefficient and prone to human error. Jewelry items are small, high-value objects that can be easily stolen or misplaced during customer trials. An automated intelligent surveillance system is required to improve security and reduce financial losses.

🛠 Technologies Used
Python
YOLOv8 (Ultralytics)
OpenCV
NumPy
Computer Vision Techniques

⚙️ Working Methodology
Capture live CCTV video frames.
Apply YOLO object detection to identify persons and objects.
Track object movement across frames.
Detect abnormal behavior such as prolonged presence or suspicious movement.
Trigger alert message in real time.

📊 Output
Bounding boxes around detected objects.
Real-time suspicious activity alert
Continuous surveillance monitoringsystem.

🚀 Future Scope
